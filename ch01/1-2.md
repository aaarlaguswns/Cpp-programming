
# 1-2 실습과제

1. IDE란 무엇인가?
	- IDE는 편집기(Editor), 컴파일러, 디버거 등 소프트웨어 개발에 필요한 모든 도구를 하나의 소프트웨어로 통합한 환경을 말합니다.
	- 복잡한 개발 과정을 한곳에서 처리할 수 있게 해주어 개발자가 효율적으로 프로그램을 제작할 수 있도록 돕는 도구입니다.
2. IDE의 종류
	- 강의노트에 나온 Visual Studio와 전 세계적으로 가장 널리 쓰이는 VS Code가 있습니다.
	- Java 개발을 위한 IntelliJ IDEA, 모바일 앱 개발용 Android Studio, Apple 생태계 전용인 Xcode 등도 유명합니다.
	- Python에 특화된 PyCharm처럼 특정 언어나 목적에 최적화된 다양한 IDE가 존재합니다.
3. IDE에 포함된 기능
	- 코드의 가독성을 높여주고 자동 완성을 도와주는 코드 편집기기능이 핵심입니다.
	- 소스 코드를 실행 파일로 바꾸는 컴파일러 및 빌드 도구와 프로그램의 오류를 찾아내고 수정하는 **디버거**가 포함됩니다.
	- 최근에는 Git 같은 버전 관리 시스템 연동이나 AI를 활용한 코드
